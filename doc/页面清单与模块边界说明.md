# 实验室知识库 + 邮箱系统页面清单与模块边界说明

## 1. 文档目的

本文档用于明确首版系统的页面范围、模块职责边界和前后端拆分原则，作为后续界面设计、接口设计和任务分配的依据。

## 2. 页面清单

### 2.0 首版入口拆分

- 公共首页：`/`，仅负责系统介绍和双入口导航
- 用户端入口：`/portal`，面向实验室普通成员
- 管理员端入口：`/admin`，面向审核、角色、群组和后续系统配置
- 旧知识库入口：`/knowledge`，保留为兼容跳转，统一重定向到用户端

### 2.1 公共页面

- 首页：系统简介、登录入口、公告与常用入口
- 注册页：成员注册、基础资料提交
- 登录页：统一登录入口，后续接 Keycloak
- 无权限页：用户无访问权限时的提示页面
- 404 页面：资源不存在提示页

### 2.2 用户中心页面

- 个人主页：个人资料展示、方向、年级、角色信息
- 资料编辑页：维护头像、昵称、联系方式等基础信息
- 我的文档页：查看我创建、编辑、收藏的知识库页面
- 我的通知页：查看系统通知、审核结果、提醒消息
- 用户端首页：用户端工作台、快捷入口、近期更新摘要

### 2.3 管理后台页面

- 后台首页：系统概览、待审核事项、统计摘要
- 用户审核页：查看注册申请并进行通过或驳回
- 用户管理页：搜索、禁用、重置状态、修改资料
- 角色管理页：维护系统角色与基础权限映射
- 群组管理页：维护方向组、年级组、功能组
- 空间管理页：维护知识库空间与可见范围
- 邮箱管理页：查看邮箱账号、邮箱开通状态、同步结果
- 审计日志页：查看关键操作记录

### 2.4 知识库页面

- 空间首页：展示空间简介、目录树、最近更新
- 目录浏览页：按目录查看页面列表
- 页面详情页：查看知识库正文、标签、附件、版本信息
- 页面编辑页：Markdown + TipTap 编辑器、预览、草稿保存
- 页面历史页：查看版本差异和回滚记录
- 页面权限设置页：配置空间继承或单页覆盖权限
- 搜索结果页：按标题、标签、内容检索文档

### 2.5 邮件协同页面

- 邮件群组页：查看方向组、年级组、通知组
- 群组映射页：配置业务群组与邮箱群组关系
- 模板管理页：维护日报、周报、通知模板
- 发送记录页：查看系统通知与邮件任务结果

## 3. 首版模块边界

### 3.1 前端模块

- `portal`：门户与公共页面
- `account`：注册、登录、个人中心
- `admin`：后台管理能力
- `knowledge`：知识库浏览、编辑、版本、搜索
- `mail`：邮箱群组、模板、发送记录
- `common`：通用布局、组件、权限提示、基础请求封装

前端职责：

- 提供用户交互界面
- 管理页面状态、表单和交互流程
- 调用后端 API 并展示结果
- 不承载核心业务规则判断
- 通过独立路由区分用户端与管理员端，降低信息噪音和误操作风险

### 3.2 后端模块

- `auth`：认证对接、登录态校验、用户上下文
- `user`：用户资料、注册审核、状态管理
- `organization`：方向、年级、群组、角色管理
- `permission`：Casbin 策略封装、资源授权校验
- `knowledge-space`：知识库空间与目录
- `knowledge-page`：页面内容、草稿、发布、版本历史
- `attachment`：附件上传、访问控制、元数据管理
- `mail-sync`：邮箱账号、群组映射、Mailcow 同步
- `notification`：站内通知、邮件任务、模板投递
- `audit-log`：审计日志采集与查询

后端职责：

- 提供统一 API 和业务规则实现
- 管理数据库、缓存、对象存储和异步任务
- 处理权限判断、审计记录和外部系统同步

### 3.3 基础设施边界

- `PostgreSQL`：业务主数据和审计数据
- `Redis + BullMQ`：缓存、异步任务和同步队列
- `MinIO`：附件和图片对象存储
- `Keycloak`：认证中心与用户身份管理
- `Mailcow`：邮箱与群组邮箱服务
- `Traefik/Nginx Proxy Manager`：统一入口和 HTTPS

基础设施职责：

- 提供通用平台能力
- 不直接承载业务决策逻辑
- 由后端统一协调使用

## 4. 前后端接口划分原则

- 前端只通过 API 访问业务能力，不直接操作基础设施组件
- 后端负责统一封装 Keycloak、MinIO、Mailcow 等外部能力
- 权限校验以后端为准，前端只做展示级拦截
- 文档正文以 Markdown 为发布态存储，编辑态可保留结构化缓存

- 页面版本历史
- 模板管理
- 审计日志查询
